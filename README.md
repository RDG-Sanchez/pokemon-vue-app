## Inicializar el proyecto:
- Clonar repositorio:
```bash
git clone https://github.com/RDG-Sanchez/pokemon-vue-app.git
```

- Dirigirse al directorio:
```bash
cd pokemon-vue-app
```

- Instalar dependencias:
```bash
npm install
```

- Iniciar servidor:
```bash
npm run dev
```

- Compilar el proyecto:
```bash
npm run build
```
